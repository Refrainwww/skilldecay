# Data Card: SkillDebtBench Artifacts

The repository includes only small derived tables and figures suitable for documentation.
Large raw rollout logs are intentionally excluded by `.gitignore`.

## Included artifacts

- `data/skilldebtbench/main_table.md`: synthetic benchmark summary.
- `data/ablation/ablation_table.md`: ablation summary.
- `data/coding_exec_skill_debt/table.md`: executable coding benchmark summary.
- `data/diagnosis/diagnosis_report.md`: dry-run diagnosis report.

## Regeneration

All raw outputs can be regenerated with scripts in `README.md`.
